# Quick Task 260508-463 — RAs do DF nas Zonas de Entrega

## Objective
Replace the free-text CityCombobox in StoreSettings zone modal with a scrollable list of all 35 Regiões Administrativas (RAs) do Distrito Federal. The list shows ALL 35 RAs — configured ones show their delivery info + edit/delete, unconfigured ones show a "Configurar" button. This makes it easy for suppliers to set up delivery zones across all DF regions.

## Context
User wants: "aparecer o nome de todas as cidades para os fornecedores preencher com a data da entrega para essas regiões e caso não entregue, ele não preencha, aparecer para o comprador para caso ele seja dessa cidade"

Translation: Show all 35 RAs. Suppliers fill in days/hours for the ones they cover. Buyers see which RAs have delivery configured.

## File to modify
- `src/pages/supplier/StoreSettings.tsx`

## Tasks

### Task 1: Add DF_RAS constant and replace zone UI with RA list

(32 unique RAs — consolidated duplicates from user's list into the official DF RAs)

2. **Change openAddZone to accept optional RA name** — when called from a RA row, pre-fill city+state:
```tsx
const openAddZone = (raName?: string) => {
  setEditingZone(null)
  setZoneForm({ city: raName || '', state: raName ? 'DF' : '', days: [], hours_start: '', hours_end: '' })
  setShowZoneModal(true)
}
```

3. **Replace the "Zonas de Entrega" section JSX** — instead of showing `zones.map(...)` after a "+ Adicionar" button, show ALL DF_RAS in a list. For each RA:
   - If a zone exists for that RA (match by `zone.city === ra`): show the zone row with edit/delete
   - If no zone: show a "Configurar" row with a "+" button to open the modal pre-filled

4. **Remove CityCombobox from the modal** — replace with a read-only display when `zoneForm.city` is pre-set, or a simpler `<input>` for manual entry if opened via the top "+ Adicionar" button. Actually: when the modal is opened from a RA row, `zoneForm.city` is already set, so show it as a read-only label. When opened via manual "+ Adicionar" (keep for custom cities), show the CityCombobox.

5. **Keep the top-level "+ Adicionar" button** for any city outside the DF RA list.

## Success criteria
